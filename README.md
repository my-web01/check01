# 入伍体检自测 H5

**基于《应征公民体格检查标准》开发的自助体检评估系统**

可直接在浏览器打开 `index.html` 使用，或部署到 GitHub Pages 获得固定网址。

---

## 功能特点

- 📏 **7大科室**：眼科、内科、外科、耳鼻咽喉、口腔、妇科（女性）、辅助检查
- 👥 **差异化标准**：支持男性/女性、普通兵/条件兵差异化评估
- 📊 **详细报告**：根据输入数据生成完整的体检评估报告
- 📋 **本地存储**：使用 localStorage，数据不上传服务器，保护隐私

---

## 体检标准差异

| 项目 | 男性普通兵 | 男性条件兵 | 女性普通兵 | 女性条件兵 |
|------|-----------|-----------|-----------|-----------|
| 身高 | ≥160cm | ≥162cm | ≥158cm | ≥160cm |
| BMI | 17.5-30 | 17.5-27 | 17-24 | 17-24 |

## 使用方式

### 方式一：直接打开

双击 `index.html` 在浏览器中打开即可使用。

### 方式二：GitHub Pages 部署

1. 将所有文件上传到 GitHub 仓库
2. 进入 Settings → Pages
3. Source 选择 `main` 分支和 `/ (root)`
4. 等待部署，获取永久网址
